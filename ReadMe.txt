What it is:

This program can be considered as a library that performs matrix and vector arithmetic.
It has multiple operations namely Scalar multiplication, Matrix multiplication and etc.
It also solves linear systems equations and performs gaussian elimination.

How it works:

Enter the matrix or vector in the required format.
Then simply read it into a matrix object and perform the operation.

"input.txt" format:
#################################################################

2 5 3              <<< This is the format needed for matrix
4 7 8
1 9 3

4 5 7 9 3 5 1 2    <<< This is the format needed for vectors

#################################################################

Note:

Please follow format strictly.
Examples of use are given.
